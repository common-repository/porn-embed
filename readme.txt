== Description ==
Adds Embed support for xhamster.com, xvideos.com, youporn.com, tube8.com and pornhub.com in WordPress posts, pages and custom post types.

== Installation ==
1. Upload the *.zip to your plugin folder / Install the Plugin via Wordpress, activate the plugin.

2. There are no settings. Simply, add the video URL in your content editor.
